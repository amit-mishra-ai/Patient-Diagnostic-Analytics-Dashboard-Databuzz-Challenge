# 🏥 Patient Diagnostic Analytics Dashboard

A **Power BI healthcare analytics dashboard** built for the **Databuzz Power BI Challenge**.  
This project transforms **10,000 patient diagnostic records** into actionable insights that help identify disease patterns, detect high-risk patients, and support data-driven clinical decision making.

### Data Tables

| Table | Description |
|------|-------------|
| **Patient_Symptoms** | Patient demographics, symptoms, fever type, risk level |
| **Blood_Test_Results** | Lab values such as WBC, RBC, Platelets, CRP, ESR |
| **Fever_Type_Profile** | Clinical reference ranges for fever types |
| **Medication** | Fever-specific treatment protocols |

## 🔍 Key Insights

- **Rash** showed the highest hospitalization rate among symptoms.
- **NLR (Neutrophil-to-Lymphocyte Ratio)** identifies early sepsis risk.
- **Platelet decline** signals potential Dengue hemorrhagic risk.
- **CRP-ESR quadrant analysis** distinguishes acute vs chronic inflammation.
- **Age 65+** group shows the highest patient visits and risk levels.
